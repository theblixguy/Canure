Canure 2.3

Welcome to Canure 2.3 Beta 2, a lightning fast and efficient web browser. It's designed to be light, quick and neat, so you'll find less distracting elements and more focus towards the web.

This is a beta preview. It's semi-tested and a little buggy. It's not ready for the big show but most of the features just work right. There might be some glitches but the overall experience is flawless

Canure 2.3 officially supports the native HTML5 and CSS3 markup. It also supports the HTML5 <audio> and <video> tags, with the support of Quicktime or Quicktime Lite.
